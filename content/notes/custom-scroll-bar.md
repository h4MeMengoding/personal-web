## CSS

```css
::-webkit-scrollbar {
  width: 0.75rem;
  height: 0.75rem;
  background-color: blue;
}

::-webkit-scrollbar-thumb {
  border-radius: 20px;
  background-color: gray;
}

/** firefox **/
html {
  scrollbar-width: thin;
  scrollbar-color: blue;
}
```
